# Clinic Management System

Aplikasi web klinik berbasis Flask untuk skripsi.

## Cara menjalankan

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python run.py
```

Buka:

```text
http://127.0.0.1:5000
```

## Akun demo

Admin: `admin@clinic.local / admin123`  
Resepsionis: `reception@clinic.local / reception123`  
Dokter: `doctor@clinic.local / doctor123`
