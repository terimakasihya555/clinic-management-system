CLINIC MANAGEMENT SYSTEM
Staff Usage Guide

1. How to Start the Application

Double-click:

Clinic Management System.exe

After that, the system will automatically open in the browser:

http://127.0.0.1:5000

3. Important Notes

- Do not delete the instance folder.
- Do not delete the logs folder.
- Do not delete the backups folder.
- Do not delete the .env file.
- The database is stored in the instance folder.
- The application log is stored in the logs folder.
- Database backup files are stored in the backups folder.

4. How to Stop the Application

Close the application window that opened when the EXE was started.

5. Recommended First Step

After first login, change the default passwords for security reasons.
